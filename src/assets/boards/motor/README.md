# Motor Images — Upload Here

Drop any `.png` motor photos into this folder.

**File names do not matter** — use any descriptive name you like, e.g.:
- `micro-5055.png`
- `outrunner-motor.png`
- `motor-close-up.png`

After committing the images and rebuilding the app, the board builder
composition box will **randomly pick one** of these images to display
whenever a motor is selected on the conveyor belt.

> **Tip:** one image per folder is enough for it to show up; add more for
> more visual variety.

See `public/assets/boards/README.md` for the full asset documentation.
