# Motor Images — Upload Here

Drop any `.png` motor photos into this folder.

**File names must include a keyword** that identifies the motor:

| Keyword in filename | Motor option   |
|--------------------|----------------|
| `micro` or `5055`  | Micro 5055     |
| `standard` or `6354` | Standard 6354 |
| `torque` or `6374` | Torque 6374    |
| `outrunner` or `6396` | Outrunner 6396 |

Example names:
- `5055-motor.png`
- `outrunner-6396.png`
- `torque-motor.png`

After committing the images and rebuilding the app, the board builder
composition box will show the correct image when its motor is selected
on the conveyor belt.

> **Tip:** one image per motor option is enough; the keyword match ensures
> the right photo appears for each selection.

See `public/assets/boards/README.md` for the full asset documentation.
